# Eatzcalc

This is a simple application to calculate nutritional values for the whole day.

## Installation

* `git clone https://github.com/wigy/eatzcalc`
* `cd eatzcalc`
* `npm install`
* `bower install`

## Running / Development

* `ember serve`
* Visit your app at [http://localhost:4200](http://localhost:4200).
